# 文化祭展示プロジェクト

本リポジトリは、文化祭展示において利用されるWebアプリケーション群の概要と導線をまとめたものです。

本プロジェクトの目的は、展示の企画そのものを定義することではなく、部活動が設計した展示体験を安定して実現するための技術基盤を提供することにあります。

---

## プロジェクト概要

本プロジェクトは、文化祭展示における部活動の企画を実現するために、Webアプリケーション群を提供するものです。

開発者は、展示そのものの企画やコンセプトを定義する立場ではなく、部活動が設計した展示体験を安定して実現するための技術基盤を提供します。

---

## プロダクト構成

本プロジェクトは以下のアプリケーションから構成されます。

### digital-ema

来場者の入力体験を受け付け、可視化するためのWebアプリケーションです。  
文化祭展示におけるデジタル入力体験の基盤として機能します。

リポジトリ：  
https://github.com/shimataiyaki/digital-ema

---

### binary-omikuji

抽選体験を提供し、紙カードと連動した出力を行うWebアプリケーションです。  
展示におけるインタラクティブな抽選体験を担当します。

リポジトリ：  
https://github.com/shimataiyaki/binary-omikuji

---

## システムの位置づけ

本リポジトリは以下の役割を持ちます。

- プロダクト群の概要提示
- 各アプリケーションへの導線提供
- 提供条件およびデータ方針への参照

実装および詳細仕様は各アプリケーションのリポジトリに分離されています。
