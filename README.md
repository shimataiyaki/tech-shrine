# 文化祭展示プロジェクト

このリポジトリは、文化祭展示のために制作されたWebアプリケーション群の概要と利用導線をまとめたものです。

## コンセプト

このプロジェクトは、
**「人の気持ちを、少しだけ外に出すための体験空間」**をテーマにしています。

現実に近いが少しだけズレた仕組みを通じて、
来場者の内面を可視化する体験を提供します。

---

## プロダクト構成

本プロジェクトは以下の2つのアプリケーションで構成されています。

### digital-ema

デジタル形式の絵馬アプリケーション
来場者が入力した内容をもとに、体験として可視化します。

→ 詳細リポジトリ：
[https://github.com/shimataiyaki/tech-shrine](https://github.com/shimataiyaki/tech-shrine)

---

### binary-omikuji

二進数を用いた抽選型おみくじアプリケーション
ランダムな数値を通じて、紙カードと連動した体験を提供します。

→ 詳細リポジトリ：
[https://github.com/shimataiyaki/tech-omikuji](https://github.com/shimataiyaki/tech-omikuji)

---
