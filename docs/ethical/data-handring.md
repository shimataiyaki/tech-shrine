# tech-shrineプロジェクト データ取り扱い方針

本方針は、tech-shrineプロジェクトにおける、文化祭展示で収集したデータの分析作業に関する開発者の約束を定めたものです。

## 第1条（定義）

本方針において、次の各号に掲げる用語の意義は、当該各号に定めるところによります。

1. **「開発者」** とは、本プロダクトの著作権者であり、本分析作業の実施主体である個人、Shimataiyakiを指します。
2. **「部活動」** とは、本プロダクトを文化祭展示のために運用する部活動を指します。
3. **「来場者」** とは、文化祭に来場し、本プロダクトを体験した個人を指します。
4. **「tech-shrineシリーズ」** （以下 **「プロダクト」** ）とは、開発者が個人で開発し、MITライセンスに基づき公開するソフトウェア群「digital-ema」および「binary-omikuji」を指します。
5. **「tech-shrineプロジェクト」** （以下 **「プロジェクト」** ）とは、開発者がプロダクトを部活動に提供し、その運用を通じて公共空間における人とテクノロジーの関わりを検証する、開発者の個人探究活動の総称を指します。
6. **「生データ」** とは、来場者がプロダクトを通じて入力した願い事の原文、および個人が特定できる可能性のある文字列を指します。
7. **「統計データ」** とは、生データを集計・数値化・抽象化した情報であり、元の文章を復元することが不可能なものを指します。

## 第2条（基本原則）

本プロジェクトにおけるデータ分析は、来場者のプライバシーと部活動との信頼関係を最大限に尊重し、以下の原則に従って行われます。

## 第3条（データの分類と取り扱い）

開発者は、収集したデータを以下のように分類し、それぞれに定められた方法でのみ取り扱います。

| データ種別 | 定義 | 取り扱い方法 |
| :--- | :--- | :--- |
| **生データ** | 願い事の原文、および個人が特定できる可能性のある文字列。 | **完全オフライン環境**でのみ処理。インターネットに接続されたAIサービスへは一切入力しない。 |
| **統計データ** | 頻出単語の集計値、カテゴリ別割合、感情スコアなど、数値化・抽象化され、元の文章を復元できない情報。 | 個人情報を含まないため、分析結果の考察や公開のためにオンラインツールを利用することを制限しない。 |

## 第4条（データの破棄）

開発者は、プロジェクトにおける分析作業の終了後、収集した全ての**生データ**を、速やかに、かつ復元不可能な形で完全に削除します。

## 第5条（結果の公開）

本プロジェクトの成果として公開される情報は、**統計データ**から導き出された考察に限定されます。願い事の原文や、個人が推測できる情報が公開されることは一切ありません。
